# 🐭 单词打地鼠游戏

一款专为小学生设计的英语单词学习游戏，通过有趣的打地鼠方式帮助孩子轻松记忆单词。

## ✨ 主要功能

### 1. 词库管理
- ✅ 内置人教版小学三到六年级上下学期单词
- ✅ 支持导入自定义单词本（txt格式）
- ✅ 单词支持多释义，出题时灵活选用

### 2. 游戏模式
- 🎮 **英→中模式**：显示英文，选择正确中文释义
- 🎮 **中→英模式**：显示中文，选择正确英文单词

### 3. 智能学习系统
- 📚 **每日学习计划**：设置每日新单词数量，循序渐进
- 🔄 **智能复习机制**：错词自动高频出现，先强化后拓展
- 📝 **错词本管理**：自动记录错词，支持"还没记住"和"彻底记住"标记

### 4. 关卡挑战系统
- 🏆 **智能计划**：根据总单词数和目标天数自动生成关卡
- 🔓 **解锁机制**：前一关正确率≥90%解锁下一关
- 👑 **星级评价**：根据正确率显示不同状态（完美通关/已通关/待提升）

### 5. 学习统计
- 📅 **学习日历**：可视化展示每天学习情况，带颜色标记
- 📊 **数据图表**：正确率趋势、掌握进度饼图、学习时长分布
- 📈 **热力图**：月度学习频率热力图

### 6. 个性化定制
- 🖼️ **更换图片**：支持自定义地鼠、锤子、逃跑、受伤图片
- 🔊 **音效开关**：可全局开启/关闭音效
- 🎨 **少儿绘本风**：蓝天白云背景，圆角按钮，可爱界面

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行游戏

```bash
python main.py
```

### 首次使用

1. 程序会自动创建 `wordlists/` 文件夹和示例单词文件
2. 将您的单词文件放入 `wordlists/` 文件夹（格式：英文,中文1,中文2）
3. 启动游戏，选择年级，开始学习！

## 📖 使用指南

### 日常学习流程

1. **选择年级**：在主菜单下拉框选择对应年级
2. **设置参数**：调整每日新单词数和每轮地鼠数
3. **选择模式**：英→中 或 中→英
4. **开始学习**：点击"开始学习"按钮
5. **单词预览**：30秒预览本轮单词，可提前开始
6. **游戏答题**：点击正确地鼠获得分数
7. **查看结果**：游戏结束显示得分和正确率

### 关卡挑战

1. 点击"关卡挑战"查看所有关卡
2. 未解锁的关卡显示🔒，需要前一关正确率≥90%
3. 点击进入关卡，完成学习和游戏
4. 达成90%正确率自动解锁下一关

### 错词复习

1. 点击"复习错词"进入错词专项练习
2. 或在错词本中标记"还没记住"增加练习次数
3. 掌握的单词可标记为"彻底记住"，不再出现

### 查看学习进度

- **学习日历**：查看每月学习打卡情况
- **学习统计**：查看正确率趋势、掌握进度等图表
- **错词本**：查看所有错词及错误次数

## 📝 单词文件格式

### 标准格式

```
apple,苹果
book,书;书本
cat,猫;猫咪
dog,狗;小狗
```

### 说明
- 每行一个单词
- 英文和中文用逗号分隔
- 多个中文释义用分号分隔
- 支持制表符分隔

## 🎯 游戏规则

### 得分机制
- ✅ 答对：+10分
- ❌ 答错：-5分（不低于0）
- 🔥 连击奖励：每连续答对5次，额外+5分

### 连击等级
- 3连击：✨ 显示
- 5连击：⚡ +5分加成
- 10连击：🔥 +10分加成

### 游戏结束
- 完成所有题目后自动结束
- 显示得分、正确率、评语
- 自动记录学习数据

## ❓ 常见问题

### Q: 如何添加自己的单词？

A: 
1. 在 `wordlists/` 文件夹中创建txt文件
2. 按格式添加单词：英文,中文
3. 重启程序自动加载

### Q: 错词太多怎么办？

A: 
1. 使用"复习错词"功能专项练习
2. 标记已掌握的单词为"彻底记住"
3. 降低每日新单词数量，打好基础

### Q: 如何更换游戏图片？

A: 
1. 点击主菜单"更换图片"按钮
2. 选择要更换的图片类型
3. 选择本地图片文件（支持png/jpg/gif）

### Q: 数据保存在哪里？

A: 所有数据保存在 `word_game.db` 数据库文件中，位于程序运行目录。

### Q: 可以多人使用吗？

A: 当前版本支持单用户，多用户需使用不同文件夹。
